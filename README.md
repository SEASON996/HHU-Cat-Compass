# 河海大学校园流浪猫管理小程序

> 一款专为河海大学三校区设计的流浪猫管理小程序，实现猫咪档案管理、地图标记、救助申请、领养登记等功能，让校园流浪猫管理更有序、更温暖。

## ✨ 核心功能

- 🗺️ **地图标记**：三校区猫咪点位可视化，支持导航
- 📋 **猫咪档案**：完整的猫咪信息管理，支持图文记录
- 🍽️ **投喂打卡**：记录投喂行为，追踪猫咪动态
- 🆘 **救助申请**：快速发起求助，实时跟进处理进度
- 🏠 **领养登记**：规范化领养流程，审核公示
- 💬 **互动留言**：猫咪专属留言区，师生交流分享
- 👑 **管理后台**：管理员审核，确保数据准确
