# 河海猫咪地图 - 技术文档

## 1. 项目简介

河海猫咪地图是一个校园猫咪打卡与信息管理平台，用户可记录偶遇或投喂猫咪的时刻，上传照片并标记位置，帮助构建校园猫咪数据库。

## 2. 技术栈

| 层级   | 技术                | 说明               |
| ---- | ----------------- | ---------------- |
| 前端   | Vue 3 + Vite      | 组合式API，响应式UI     |
| 前端UI | Vant 4            | 移动端组件库           |
| 状态管理 | Pinia             | 用户登录状态等          |
| 后端   | Python FastAPI    | 异步高性能API框架       |
| 数据库  | PostgreSQL        | 关系型数据库           |
| ORM  | SQLAlchemy 2.0    | 数据库操作            |
| 认证   | JWT (python-jose) | 用户登录令牌           |
| 文件存储 | 本地文件系统            | 上传图片存储于 uploads/ |

## 4. 数据库准备

### 4.1 启动PostgreSQL服务

- **Windows**: 服务中启动 `postgresql-x64-xx`
- **Mac**: `brew services start postgresql`
- **Linux**: `sudo systemctl start postgresql`

### 4.2 创建数据库

```sql
CREATE DATABASE hhu_cat_db;
```
### 4.3 表结构（SQLAlchemy 模型）

```python
# 核心表
- users      # 用户表（学号、昵称、角色）
- cats       # 猫咪表（基本信息、位置坐标、状态）
- records    # 打卡记录表（关联用户和猫咪）
```
### 4.4 数据库迁移（Alembic）

```bash
# 初始化 Alembic
alembic init alembic

# 生成迁移脚本
alembic revision --autogenerate -m "init tables"

# 执行迁移
alembic upgrade head
```

## 5. 项目安装与运行
### 5.1 克隆项目

```bash
git clone <仓库地址>
cd HHUCatCompass
```
### 5.2 后端配置与启动

安装依赖:

```bash
cd backend

# 创建虚拟环境（推荐）
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
```

编辑 `.env` 文件，修改数据库连接：

```env
# 数据库连接
DATABASE_URL=postgresql://用户名:密码@localhost:5432/hhu_cat_db

# JWT 密钥（生产环境需更换）
SECRET_KEY=your-secret-key-min-32-bytes

# 文件上传目录
UPLOAD_DIR=uploads
```

数据库迁移:

```bash
# 初始化 Alembic（首次运行）
alembic init alembic

# 生成迁移脚本
alembic revision --autogenerate -m "init tables"

# 执行迁移建表
alembic upgrade head
```

启动后端：

```bash
uvicorn app.main:app --reload --port 8000
```

### 5.3 前端配置与启动

```bash
cd frontend

# 安装依赖
pnpm install

# 启动前端
pnpm run dev
```

访问：[http://localhost:5173](http://localhost:5173/)

## 7. API 接口概览

| 模块  | 方法   | 路径                        | 说明          |
| --- | ---- | ------------------------- | ----------- |
| 认证  | POST | `/auth/login`             | 用户登录        |
| 认证  | POST | `/auth/register`          | 用户注册        |
| 猫咪  | GET  | `/cats/`                  | 获取猫咪列表      |
| 猫咪  | GET  | `/cats/{cat_id}`          | 获取猫咪详情      |
| 记录  | POST | `/records/`               | 创建打卡记录      |
| 记录  | GET  | `/records/user/{user_id}` | 获取用户打卡列表    |
| 记录  | GET  | `/records/cat/{cat_id}`   | 获取猫咪打卡记录    |
| 上传  | POST | `/upload/image`           | 上传图片（返回URL） |

详细交互可查看 FastAPI 自动生成的 `/docs` 页面。
